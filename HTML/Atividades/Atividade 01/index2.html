Curiosidades

O CERN é o berço da World Wide Web. Foi lá, em 1989-1990, que o físico Tim Berners-Lee inventou o HTML, o HTTP e o primeiro navegador/servidor para facilitar o compartilhamento de informações entre cientistas. O HTML foi criado como a linguagem base para estruturar esses documentos no sistema web pioneiro
